# Dooray 업무 댓글

사용자가 준 댓글 내용을 임의로 다듬지 않는다. 셸 이스케이프와 인코딩 문제를 막기 위해 댓글을 임시 파일에 그대로 기록하고, 댓글 내용을 셸 명령에 직접 삽입하지 않는다.

저장소 루트에서 실행한다.

```text
python Dooray/dooray.py comment 업무번호 --file 임시파일경로
```

댓글은 API 토큰 소유자 명의로 등록된다. 명령 실행 후 생성한 임시 파일을 제거한다.
