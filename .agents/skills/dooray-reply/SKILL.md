# Dooray 업무 댓글

사용자가 준 댓글 내용을 임의로 다듬지 말고 그대로 스크래치패드 임시 파일에 쓴다. 셸 명령에 댓글 내용을 직접 삽입하지 않는다.

저장소 루트에서 다음 명령을 실행한다.

```text
python Dooray/dooray.py comment 업무번호 --file 임시파일경로
```

댓글은 API 토큰 소유자 명의로 등록된다. 명령 실행 후 생성한 임시 파일을 제거한다.
